## Project Backgroud
- 개요
  실시간 웹캠 영상을 통해 운전자의 졸음운전 여부를 판별하는 프로그램의 성능 향상에 대한 연구
- 필요성
  화물 및 운송 차량 운전자의 상태를 실시간으로 확인함으로써 졸음운전으로 인한 사고를 예방하고 운전자에게 안전을 제공함
